# bookbot

BookBot is a simple book analysis tool as a Python learning project from [Build a Bookbot in Python \[Full Course\] | Boot.dev](https://www.boot.dev/courses/build-bookbot-python).

![course completion badge](https://qvault-webapp-dynamic-assets.storage.googleapis.com/certificates/b3ad8388-316a-44e3-b3e1-e69082546dcb.jpeg?v=1769639674)

## Usage

1. Ensure you have Python3 installed
2. Download books as plain text files such as from [Project Gutenberg](https://www.gutenberg.org/) and store them in a `book/` subdirectory. Examples:
   - [books/frankenstein.txt](https://storage.googleapis.com/qvault-webapp-dynamic-assets/course_assets/frankenstein.txt)
   - [books/mobydick.txt](https://storage.googleapis.com/qvault-webapp-dynamic-assets/course_assets/mobydick.txt)
   - [books/prideandprejudice.txt](https://storage.googleapis.com/qvault-webapp-dynamic-assets/course_assets/prideandprejudice.txt)
3. Run `python3 main.py <path_to_book>`
